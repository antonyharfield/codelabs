เรื่อง

กระบวนการคิดเพื่อการเขียนโปรแกรมและการรับค่าผ่านทางคีย์บอร์ด

วัตถุประสงค์

เพื่อฝึกฝนกระบวนการคิดในการพัฒนาโปรแกรม และการรับค่าผ่านทางคีย์บอร์ด

หัวข้อ

  • ทบทวนการประกาศร่วมกับกำหนดค่าตัวแปร และการใช้งานตัวดำเนิน
  • ฝึกการใช้งานการแสดงผลทางจอภาพ และการรับค่าผ่านทางคีย์บอร์ด

กรณีต้องการรับค่าเป็น

  • String String s = sc.nextLine();
  • int int x = sc.nextInt();
  • float float y = sc.nextFloat();
  • double double z = sc.nextDouble();

ก่อนรับค่า

  1. ที่ด้านบนของไฟล์ import java.util.Scanner;
  2. ภายใน main Scanner sc = new Scanner(System.in);

จุดสำคัญ

  • สร้างเพียง 1 Scanner ในโปรแกรม
  • ตัวแปร sc สามารถเปลี่ยนชื่อได้
  • อย่าผสม nextLine กับ nextInt/Float/Double ในโปรแกรมเดียวกัน

HelloName.java

import java.util.Scanner;

class HelloName {

    public static void main(String[] args) {

        System.out.print("What is your name? ");

        Scanner sc = new Scanner(System.in);
        String name = sc.nextLine();

        System.out.println("Hello " + name);
    }    
}

กรณีต้องการรับค่าเป็น String

String s = sc.nextLine();
