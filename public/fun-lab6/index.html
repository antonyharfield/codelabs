หัวข้อ

  • ฝึกฝนการใช้คำสั่ง for ซ้อน for
  • ฝึกฝนการใช้คำสั่ง for กับ if

Task 1

class Task1 {

   public static void main(String[] args) {
      for (int i = 1; i <= 3; i++) {
         for (int j = 2; j <= 4; j++) {
            int x = i + j;
            System.out.println(x);
         }
      }
   }
}

จงเขียนผลลัพธ์ของโปรแกรมในตารางต่อไปนี้

ค่าตัวแปร i

ค่าตัวแปร j

แสดงผลทางจอภาพ
