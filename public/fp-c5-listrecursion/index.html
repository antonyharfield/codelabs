What you will do

In this codelab, you will write recursive functions for list operations.

Lists

A list is made from "con cells".

  • The "head" is the first element of a list
  • The "tail" is the rest of the list (not including the first element)

Example

val list = List(2, 3, 4)

val updated = 1 :: list // List(1, 2, 3, 4)

val head :: tail = list // 2 :: List(3, 4)

Recursion

A recursive function is a function that calls itself.

  • "Base case" is when the function no longer calls itself - must have at least 1 base case!
  • "Recursive case" is when the function calls itself.

Example

def sumToZero(n: Int): Int =
  if (n == 0) 0
  else n + sumToZero(n - 1)

def sumL(l: List[Int]): Int =
  l match {
    case Nil => 0
    case head::tail => head + sumL(tail)
  }

Think about a list:

  • Length of empty list is 0
  • Length of head is 1
  • Length of tail is... recursive call!
