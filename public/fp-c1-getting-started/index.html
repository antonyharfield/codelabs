There are many ways to start building Scala programs. One of the easiest is to use an IDE. IntelliJ is an IDE by JetBrains that supports Java, Kotlin, Scala and many other languages. We will be using IntelliJ throughout the course.

Installation

We will install the IntelliJ IDEA Community Edition. It is free and open-source. It works on Windows, macOS and Linux. Start by downloading and installing IntelliJ IDEA.

Download IntelliJ IDEA

Loops on lists

In Java, we would have defined an array as int[] numbers = {1, 1, 2, 3, 5, 8}; and used a for loop to print out the numbers. We can do the same in Scala using lists.

There are shorter and easier ways to loop a list in Scala. Try these:

DeepDive.sc

val numbers = List(1, 2, 3, 4, 5, 6)

for (n <- numbers) {
  println(n)
}

numbers.foreach(n => {
  println(n)
})

numbers.foreach({
  println(_)
})

numbers.foreach(println(_))

numbers.foreach(println)

Filtering a list

Now we understand that numbers.foreach(println) will print a list of numbers. What do these statements do?

DeepDive.sc

val numbers = List(1, 2, 3, 4, 5, 6)

numbers
  .filter(n => n < 3)
  .foreach(println)

numbers
  .filter(_ > 2)
  .foreach(println)

numbers
  .filter(_ % 2 == 0)
  .foreach(println)

numbers
  .filter(_ % 2 == 0)
  .filter(_ > 2)
  .foreach(println)

The filter function removes items from a list.

Modifying list items

The map function modifies each item of a list.
